# User Manual

## Shortcuts

---

> 打开文件: `Ctrl + O`
>
> 打开列表: `Ctrl + L`
>
> 切换模式: `Ctrl + M`
>
> 上一曲/下一曲: `Ctrl + Left/Right`
>
> 切换暂停/开始: `Space`
>
> 快捷切歌: 双击列表元素
